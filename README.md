# AdventureWorks Data Cleaning and Manipulation Project

## 📌 Project Overview

This project demonstrates my ability to clean, transform, and prepare raw business datasets for analysis using the AdventureWorks dataset.

The objective was to convert messy and inconsistent data into a structured, analysis-ready format that can be used for business intelligence, reporting, and advanced analytics.


## 📂 Dataset

The project consists of two versions of the dataset:

* **Raw Dataset:** Original AdventureWorks data containing inconsistencies and formatting issues.
* **Cleaned Dataset:** Processed and standardized dataset ready for analysis.

The datasets include information on:

* Customers
* Products
* Sales
* Returns
* Sales Territories
* Product Categories and Subcategories


## 🧹 Data Cleaning and Transformation Tasks

The following operations were performed during the cleaning process:

* Standardized column names
* Removed duplicate records where necessary
* Corrected inconsistent data formats
* Renamed columns for clarity and consistency
* Checked for missing values and handled them appropriately
* Organized tables for easier analysis
* Ensured data integrity across related tables
